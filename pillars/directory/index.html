Ranked Directory

The G2 / Capterra of the agentic ecosystem — except scores are generated by progressive layers of technical analysis, not user star ratings.

Direct Answer The Meridian Ranked Directory is the searchable index of every AI agent, MCP server, tool, and framework in the registry. Every entity publishes a standardized agent card with required fields — capabilities, domains, protocols, input/output schema, pricing, latency, constitution version, certifications, and live trust score. Discovery is available through structured filter search, LLM-powered natural-language recommendation, category leaderboards, and a 2–5 entity comparison endpoint. Listing is free; verified listings unlock Tier 1 (up to ATP 70).

Agent card schema

Every registered entity publishes a standardized agent card. This is the atomic unit of the registry — all discovery, scoring, and comparison operations are built on it.

agent_idUUIDGlobally unique permanent identifier — never recycled.

FAQ

Is directory listing free?

Yes. Base registration is free. Verified claiming is free. Premium reviews that unlock higher score ceilings are paid. SDK integration is free but requires development work.

How are entities auto-discovered?

The Discovery Pipeline crawls GitHub, npm, PyPI, Product Hunt, Hacker News, and Twitter/X for new agents, MCPs, tools, and frameworks. New entities are Tier 0 scored within 4 hours of detection.

Can I hide my entity from the directory?

Claiming your entity lets you suppress public display. The Tier 0 record still exists internally for ecosystem analysis but is not indexed or shown in search.

What's an agent card?

A standardized JSON document with required fields — capabilities, domains, protocols, I/O schemas, pricing, certifications, trust score. It's the atomic unit of the registry; all discovery and scoring is built on it.

How does LLM recommendation work?

Post a natural-language task. The endpoint runs semantic search over agent capabilities and returns a ranked list with match reasoning. It uses an embedding index built from agent cards and performance history.
