The Meridian SDK v2.0 is the open-source middleware that Tier 3 entities integrate for live telemetry. Available in Python, JavaScript/TypeScript, and Go.

Privacy guarantees are non-negotiable and enforced in the open-source implementation: no task content ever transmitted, no user data transmitted, audit logs stay local (only hash chains transmitted for integrity verification), full source on GitHub, no obfuscated code, no binary blobs.

Seven modules: Telemetry, Compliance Monitor, Security Monitor, Audit Trail, Compliance Endpoint, Durability Monitor, Cost Tracker. See the API reference for full module details.
